1. Напишите функцию, которая принимает список чисел и возвращает сумму всех элементов списка.  
2. Напишите функцию, которая принимает список и возвращает новый список, содержащий только чётные числа из исходного.  
3. Напишите функцию, которая принимает список строк и возвращает самую длинную строку из него. Если список пуст, верните пустую строку.  
4. Напишите функцию, которая принимает список и число, и возвращает True, если это число встречается в списке ровно один раз.  
5. Напишите функцию, которая принимает список чисел и возвращает список из их квадратов (например, [1, 2, 3] → [1, 4, 9]).  
6. Напишите функцию, которая принимает список и возвращает новый список с элементами в обратном порядке (без использования метода reverse() или среза [::-1]).  
7. Напишите функцию, которая принимает список строк, каждая из которых представляет собой целое число (например, ["1", "20", "-5"]), и возвращает новый список целых чисел, преобразованных из этих строк. Игнорируйте строки, которые нельзя преобразовать в целое число.  
8. Напишите функцию, которая принимает список строк (некоторые из них могут быть числами в виде строк, другие — обычными словами). Функция должна:  
- отфильтровать только те строки, что можно превратить в целые числа, и преобразовать их в числа (используйте функцию из задачи 7),  
- вычислить сумму этих чисел (функция из задачи 1),  
- оставить только чётные из них (функция из задачи 2),  
- возвести их в квадрат (функция из задачи 5),  
- развернуть полученный список (функция из задачи 6),  
- проверить, встречается ли число 4 в этом списке ровно один раз (функция из задачи 4),  
- из исходного списка строк выделить только те, что не являются числами, и найти среди них самую длинную строку (функция из задачи 3).  
Верните кортеж из трёх элементов: (сумма всех чисел, развернутый список квадратов чётных чисел, самая длинная нечисловая строка).  
Всё должно быть реализовано с использованием ранее написанных функций (1–7).
