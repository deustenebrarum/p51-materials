Вопрос 5. Можно ли использовать стрелочную функцию в качестве функции-конструктора?

  this.model = model;
};
const myCar = new Car("BMW");

A) Да, myCar будет объектом { model: "BMW" }.
B) Нет, будет выброшена ошибка TypeError: Car is not a constructor.
C) Да, но нужно обязательно вернуть this из функции.
D) Нет, код выполнится без ошибок, но myCar будет undefined.
