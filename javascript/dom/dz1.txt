Советы по выполнению:
- Чтобы найти элемент, используйте команду: const имя = document.querySelector("тег");
- Пишите весь код внутри тега script, который должен находиться в самом низу вашего HTML-файла.
- Каждая задача независима: для каждого пункта лучше создать свой отдельный тег в HTML.

Задание 1. Изменение текста
Пояснение: Свойство textContent позволяет добраться до букв внутри тега и заменить их на новые. Это самый простой способ обновить информацию на странице.
Задача: Создайте в HTML заголовок h1 с любым текстом. В JavaScript найдите этот заголовок и замените его текст на слово "Привет".

Задание 2. Изменение стиля
Пояснение: Свойство style дает доступ к CSS-правилам элемента. С его помощью можно менять цвета, размеры и отступы прямо во время работы программы.
Задача: Создайте в HTML обычный текст (тег p). В JavaScript найдите его и через свойство style.color измените цвет этого текста на красный.

Задание 3. Реакция на клик
Пояснение: Метод addEventListener ("слушатель событий") заставляет элемент "ждать", пока пользователь совершит какое-то действие, например, нажмет на него мышкой.
Задача: Создайте кнопку (button). Напишите код, который при клике на эту кнопку изменит ее собственный текст на "Нажато!".

Задание 4. Изменение атрибутов
Пояснение: Метод setAttribute позволяет менять настройки тега, которые прописаны внутри него. Например, можно изменить адрес, куда ведет ссылка, или путь к картинке.
Задача: Создайте в HTML пустую ссылку (тег a) без адреса. В JavaScript найдите эту ссылку и через setAttribute добавьте ей адрес "https://ya.ru".

Задание 5. Добавление нового элемента
Пояснение: Команда appendChild позволяет взять любой созданный вами элемент и "приклеить" его в конец страницы или внутрь другого тега.
Задача: Создайте в JavaScript новый элемент (например, кнопку или текст) через createElement, добавьте в него любой текст и с помощью appendChild прикрепите его в самый конец вашего документа (в document.body).
