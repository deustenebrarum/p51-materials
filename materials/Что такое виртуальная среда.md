Среда создаётся командой:
```sh
python -m venv venv
```

Среда является отдельной копией питона прямо в папке нашего проекта, которая будет содержать необходимые версии библиотек для этого проекта.
Чтобы активировать среду
```sh
./venv/Scripts/activate
```
Деактивировать среду:
```sh
deactivate
```

После активации среды слева от статусной строки терминала появляется название папки со средой в скобках(например: `(venv)`)
Все остальные операции с python и pip будут происходить только в этой папке.
