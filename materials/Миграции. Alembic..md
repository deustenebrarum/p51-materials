Инструмент alembic занимается миграциями SQLAlchemy. Он помогает синхронизировать последовательно все изменения в базах данных на всех устройствах.
 Установка
 ```sh
 pip install alembic
 ```
 Инициализация:
 ```
 alembic init alembic
 ```
 В новосозданном файле `alembic.ini` нужно в строке с параметром `sqlalchemy.url` прописать строку подключения.
 ```python
 sqlalchemy.url = postgresql://postgres:admin@localhost/p51_products_test
 ```
 В `alembic/env.py` импортировать все модели, а так же `Base.py` и прописать параметр 
 ```python
 target_metadata = Base.metadata 
 ```
